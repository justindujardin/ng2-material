Introduction

Examples of how to use various material design inspired components with angular2.

There are a number of community developed components available in the ng2-material package.

There are also a number of examples demonstrating the use of the official Google developed material2 packages.

Installing from NPM

Install the ng2-material and the material2 core libraries:

npm install --save ng2-material @angular/material

Then import the material directives and providers

import {MATERIAL_DIRECTIVES, MATERIAL_PROVIDERS} from "ng2-material";

Then reference the styles in your page

<link rel="stylesheet" href="node_modules/ng2-material/ng2-material.css"> <link rel="stylesheet" href="node_modules/ng2-material/font/font.css">

Using Material2 Components

To make use of the official google material2 components you will need to install them alongside ng2-material as needed. ng2-material has peerDependencies on the components that it uses internally. The various examples of official components describe how to install them.
