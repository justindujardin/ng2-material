Installing from NPM

If you are a ninja, feel free to npm install it

npm install --save ng2-material

Then import the material directives and providers

import {MATERIAL_DIRECTIVES, MATERIAL_PROVIDERS} from "ng2-material/all";

Then reference the styles in your page

<link rel="stylesheet" href="node_modules/ng2-material/dist/ng2-material.css"> <link rel="stylesheet" href="node_modules/ng2-material/dist/font.css">

Or (more preferably) import them into your SCSS files.

$md-font-url: "node_modules/ng2-material/dist/"; @import "node_modules/ng2-material/dist/font"; @import "node_modules/ng2-material/source/all";

If you want to use custom theme colors, include the core styles and override the theme variables before including the components.

$md-font-url: "node_modules/ng2-material/dist/"; @import "node_modules/ng2-material/dist/font"; @import "node_modules/ng2-material/source/core"; $md-primary: md-palette($md-brown, 500, 100, 700, $md-contrast-palettes); $md-accent: md-palette($md-deep-orange, 500, 300, 700, $md-contrast-palettes); @import "node_modules/ng2-material/source/components";
